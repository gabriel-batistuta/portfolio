Anime Seasonal

um algoritmo que usa a API do My Anime List pra pegar os animes da temporada atual e transformar em dados em um pdf

Code Link

Visão Geral do Projeto

O objetivo do projeto é fazer um algoritmo que você possa usa-lo no inicio de cada temporada de anime e ele te dará um arquivo pdf com todos os novos animes e suas imagens, descrições, gêneros e nome de studio.

Foi usada a API do MAL para pegar as informações dos animes e a API do Google Tradutor para traduzir as informações vindas da api. O algoritmo gera o pdf a partir de um template html e css com o formato estruturado de um anime por página do pdf.
