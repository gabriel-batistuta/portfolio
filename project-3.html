DxDFC Reader

Esse é um código feito pra criar um arquivo pdf do volume de uma light novel encontrada na internet automaticamente

Code Link

Visão Geral do Projeto

O objetivo do projeto é criar um arquivo pdf para leitura de uma light novel encontrada na internet em forma de site. Também foi necessário filtrar os dados requeridos, baixar a capa do volume para a pasta da novel e gerar um índice em formato .txt dos capítulos do volume o usuário consultar.

Para fazer o projeto foi necessário Web Scraping e Web Crawllers para a busca e extração dos dados no site. Após a extração e filtragem dos dados, é gerado o pdf a partir de um template html e css com o formato estruturado sequencialmente de um titulo principal do capitulo atual, dia de lançamento e o texto da novel com as ilustrações.
