Steam Wrapper API

Uma biblioteca da API da Steam feita em Python que facilita seu uso

Link da biblioteca

Visão Geral do Projeto

O projeto visa simplificar o uso da API da Steam para o Python, a biblioteca feita cria um objeto Steam que pode pegar as informações da API da Steam sem que o usuário precise fazer a requisição http da api ou converter o objeto JSON pra objeto Python, precisando apenas que o usuário dê os parâmetros requeridos para pegar as informações.

O projeto foi publicado no PIP e está disponível para uso público.
